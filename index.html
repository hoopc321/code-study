프로그램 구조

예제

#include<stdio.h>

int main(){


     return 0;
}


C언어의 가장 기본적인 구조이다.
#include<stdio.h>를 통해 stdio.h라는 헤더파일을 불러와 여러 함수를 사용할 수 있도록 하는 것이다.
또, C언어는 int main()을 사용하여 main이라는 정수형 함수를 선언, 함수의 리턴값을 운영체제가 가장 먼저 받아들이게 한다.
return 0는 0의 값을 반환한다는 뜻이고 메인함수에서는 코드를 끝내는 역할로 사용된다.
C에서는 모든 구문이 끝날 때 ;(세미콜론)을 사용하여 코드들을 구분하며 영어의 대소문자를 구분한다.
